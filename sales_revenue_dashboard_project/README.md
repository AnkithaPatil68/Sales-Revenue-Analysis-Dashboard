
Sales & Revenue Analysis Dashboard Project

Files:
- sales_data.csv : Sample dataset
- sales_dashboard.xlsx : Excel file containing the dataset

Steps to Build Dashboard:
1. Open the Excel file.
2. Insert Pivot Tables for:
   - Total Revenue
   - Sales by Category
   - Sales by Region
3. Create charts:
   - Line Chart → Revenue Trend by Date
   - Bar Chart → Top Products
   - Pie Chart → Region-wise Sales
4. Add slicers for:
   - Region
   - Category
   - Product

KPIs to include:
- Total Sales
- Total Revenue
- Top Performing Product
- Average Order Value
